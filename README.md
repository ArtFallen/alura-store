# Análisis de rendimiento — Alura Store

Proyecto enfocado en evaluar el desempeño comercial de cuatro tiendas para decidir cuál sucursal conviene vender y así financiar un nuevo emprendimiento.

## Objetivo
Comparar indicadores de ventas y satisfacción para detectar la tienda con menor eficiencia operativa.

## Análisis realizado
- Ingresos totales por tienda
- Ventas por categoría de producto
- Calificaciones promedio de clientes
- Productos con mayor y menor rotación
- Costos promedio de envío
- Visualizaciones comparativas

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab / Jupyter Notebook

## Resultado
El estudio permite identificar la tienda con menor rendimiento relativo, apoyando la toma de decisiones mediante evidencia cuantitativa y gráficos comparativos.

## Uso
1. Cargar los archivos CSV de ventas.
2. Ejecutar el notebook de análisis.
3. Revisar gráficos y conclusiones finales.

## Propósito académico
Proyecto orientado a practicar limpieza de datos, visualización y análisis para decisiones comerciales basadas en información real.
